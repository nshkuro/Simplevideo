Модуль позволяет добавлять в контент поле с практически любым флеш-плеером,
загружать видео, пережимать его с нужными параметрами, создавать превьюшки в любом колличестве.
Принцип работы в следующем:
    1) Необходимо создать пресет (в меню по адресу admin/build/videofield). В пресете выбирается действие которое он будет выполнять(пережимать видео или делать превьюшки).
       Стоит заметить в пресете можно задать только одно дейсвие!!! Также задается режим как это будет выполняться, по крону или
       сразу при обращении к файлу. Рекомендуется пережимать видео по крону, а генерить превьшки сразу при обращении.
	   Внимание: Вы не сможете использовать крон, если у вас не установлен дополнительный модуль videofield_cron.
    2) Далее создается плеер (в том же меню что и пресет), на основе которого будет создан форматер к FileField полю в которое будет загружаться видео.
       При создании плеера указывается путь к нему, машинное имя форматтера, и человеко-понятное имя форматера, а так же
       указываются параметры плеера и переменные. В переменных в виде токенов задаются пресеты для видео и превьюшек.
	   ПРИМЕР: с модулем поставляется viral-player. Рассмотрим пример на его основе.
			1) Укажем путь до плеера относительно папки с проектом (если проект лежит в папке var/www, 
			то путь будет sites\all\modules\custom\videofield\players\mediaplayer-viral\player-viral.swf, 
			если проект находится в var/www/poject, то путь будет project\sites\all\modules\custom\videofield\players\mediaplayer-viral\player-viral.swf)
			2) Укажем параметры для плеера (смотри в документацию к плееру) здесь это будет:
			width 470
			height 320
			allowfullscreen TRUE
			3) Укажем переменные к плееру (какие существуют смотри докумнтацию к плееру)  (снизу в таблице доступных токенов виден список пресетов)
			например file [video1]
					 img [thumb1]
					 type [video]
			4) Все готово. Жмем сохранить.
    3) В displayfield свойствах поля filefield задется нужный форматер с плеером.
    4) При необходимости(для windows или более экзотических случаев) задать путь к ffmpeg.exe
    5) При необходимости можно посмотреть список выполненых заданий и задания в очереди (admin/reports/videofieldcron), на странице выводится список их 30 значений.
